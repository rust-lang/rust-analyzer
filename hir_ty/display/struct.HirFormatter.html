The sink to write into

§buf: String

A buffer to intercept writes with, this allows us to track the overall size of the formatted output.

§curr_size: usize

The current size of the formatted output.

§max_size: Option<usize>

Size from which we should truncate the output.

§entity_limit: Option<usize>

When rendering something that has a concept of “children” (like fields in a struct), this limits how many should be rendered.

§show_container_bounds: bool

When rendering functions, whether to show the constraint from the container

Whether formatting impl Trait1 + Trait2 or dyn Trait1 + Trait2 needs parentheses around it, for example when formatting &(impl Trait1 + Trait2).
