Renders a [SyntaxNode] with whitespace inserted between tokens that require them.

This is an internal API that is only exported because mbe needs it for tests and cannot depend on hir-expand. For any purpose other than tests, you are supposed to use the prettify_macro_expansion from hir-expand that handles $crate for you.
