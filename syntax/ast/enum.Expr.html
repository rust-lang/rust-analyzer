pub fn parse(text: &str, edition: Edition) -> Parse<Expr>

Parses an ast::Expr from text.

Note that if the parsed root node is not a valid expression, Parse::tree will panic. For example:

ⓘ
ast::Expr::parse("let fail = true;", Edition::CURRENT).tree();
