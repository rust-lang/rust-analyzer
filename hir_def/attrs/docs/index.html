Documentation extraction and source mapping.

This module handles the extraction and processing of doc comments and #[doc = "..."] attributes, including macro expansion for #[doc = macro!()] patterns. It builds a concatenated string of the full docs as well as a source map to map it back to AST (which is needed for things like resolving links in doc comments and highlight injection).
