Makes an optional return type along with whether the return type should be focused by the cursor. If we cannot infer what the return type should be, we create a placeholder type.

The rule for whether we focus a return type or not (and thus focus the function body), is rather simple:

  • If we could not infer what the return type should be, focus it (so the user can fill-in the correct return type).
